# IPL-WINNING-TEAM-PREDICTION
IPL Winning Team Prediction is a machine learning project that forecasts the IPL champion using team form, player stats, net run rate, and venue effects. It applies ensemble classification models to output win probabilities, helping fans and analysts compare contenders in real time.
